AI_USAGE.md

- We used AI to help write most of the initial code for the app (different pages like upload, cleaning, visualization, and export).
- We also used AI when we had errors, to understand the problem and get possible fixes.
- All code was checked and tested manually. We made sure the app works without errors, the data transformations give correct results, the charts show correctly
- If something didn’t look good (especially design or layout), we fixed it ourselves by changing specific lines in the code.
- We manually improved things like colors, layout, and text visibility in dark/light mode so everything looks better.
- We tested the app with different datasets (including missing values and duplicates) to make sure everything works properly.
